# Hexagon Music Visualizer README

Create stunning hexagon-based music visualizations with this interactive web app. It analyzes audio input in real-time to generate dynamic, colorful patterns deployed via GitHub Pages.[1]

## Live Demo
Try it live here: [Hexagon Music Visualizer](https://layasri157.github.io/hexagon-music-visualizer/)[1]

## Features
- Real-time audio visualization using Web Audio API.
- Hexagonal particle effects that pulse to the beat.
- Responsive design works on desktop and mobile.
- Customizable colors and sensitivity settings.

## Quick Start
1. Clone the repo: `git clone https://github.com/layasri157/hexagon-music-visualizer.git`
2. Open `index.html` in a modern browser (Chrome/Firefox recommended).
3. Grant microphone access when prompted.
4. Play music and watch the hexagons dance!

## Tech Stack
- HTML5 Canvas for rendering.
- Web Audio API for frequency analysis.
- Vanilla JavaScript for smooth 60fps animations.

## Deployment
Already deployed via GitHub Pages at `/settings/pages`. Source branch: `main`.[1]
